# Temperature Converter

A simple web-based temperature converter built using HTML, CSS, and JavaScript. The application allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin through an easy-to-use interface.

## Features

- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- Simple and responsive user interface
- Instant temperature conversion

## Technologies Used

- HTML5
- CSS3
- JavaScript
